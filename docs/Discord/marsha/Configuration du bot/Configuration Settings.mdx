## 1.1 - Guilds Settings

Dans cette page de configuration lié à votre serveur vous allez retrouvez plus chose:

Vous allez pouvoir modifié le **prefix** qui sera utilisé pour les commandes. (Par default: `!`),<br/>
Changer la langue du bot pour quelle sois adapter au serveur. (Par default: `English`).

![](https://media.tutorapide.xyz/h2hx39nv5hge.png)

Dans la partie Request, vous allez retrouver les informations que vous avez fournit lors de votre inscription.

Un email , qui vous sert a comunniquez avec l'équipe de Marsha.<br/>
Le lien d'inviitation de votre serveur, pensez a le mettre a jour si vous avez supprimer les inviitations sur votre serveur.<br/>
Une partie description , qui présente votre serveur.

![](https://media.tutorapide.xyz/2owphgekqa9c.png)

## 1.2 - Moderation Settings

Voyons dans un premier temps la partie Modération qui est composé de **4 catégories**:

* Moderation
* Message limits
* Badwords
* Automated Warnings

### La premiére catégorie Moderation

La premiére sous-catégorie **Domains**.

Va vous permettre d'autoriser des lien provent de "source sur".

La deuxieme sous-catégorie **Invites**.

Va vous permettre de whitelist les invites de certains serveur discord.

![](https://media.tutorapide.xyz/xa1jwbsm3aqa.png)

### La deuxiéme catégorie Message limits

La sous-catégorie **Max Mentions**.

Vous permet de definir le nombre de mentions maximum par message.

La sous-catégorie **Max Emojis**.

Le nombre de emoji maximum par message.

![](https://media.tutorapide.xyz/1n1mmtjij0g4.png)

### La troisiéme catégorie Badwords

Cette catégorie va vous permettre de mettre une liste de mot en blacklist , comme des insultes.

![](https://media.tutorapide.xyz/jklo4azgbpko.png)

### La quatriéme catégorie Automated Warnings

Cette catégorie va nous permettre de configurer les avertissements automatique. Et ainsi definir les actions que nous souhaitons effectuer.

![](https://media.tutorapide.xyz/2xtqv5npuy8k.png)

## 1.3 - Moderation Groups

Dans cette catégories nous allons pouvoire definir différent groupes pour que le bot puisse savoir comment géré la Moderation.

Vous allez pouvoir créée plusieurs groupes: 

    **Staff**
    **Membres**

La création d'un groupe va nous permettre ainsi de configurer l'auto mods, pour nous permettre de créée des whitelist.

![](https://media.tutorapide.xyz/uxjaibs97ywj.png)

## 1.4 - Moderation Report

Dans cette catégorie nous allons pouvoir mettre un place un systeme de report.
Ce systeme peut etre utiles si vous souhaitez mettre un place un systeme d'aide ou de report
